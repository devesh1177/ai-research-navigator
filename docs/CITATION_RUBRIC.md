\# Citation Faithfulness Rubric



\## Criterion 1: Citation Coverage



Does every factual claim contain a citation?



Score:

\- 0 = No

\- 1 = Partially

\- 2 = Yes



\## Criterion 2: Source Support



Does the cited source support the claim?



Score:

\- 0 = Unsupported

\- 1 = Partially Supported

\- 2 = Fully Supported



\## Criterion 3: Uncited Claims



Are there uncited factual claims?



Score:

\- 0 = Many

\- 1 = Few

\- 2 = None



Maximum Score = 6
